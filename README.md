# Fetch Me Some Fake Planets
Exercise retrieving and sorting planet data from the SWAPI. Viewable at http://andreakereliuk.com/fetch-swapi/

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Setup

When starting new, first run `npm install`. 
To run the dev environment & compiler, run `npm start`
To build for production, run `npm run build`

## Folder Structure

```
app/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    components/
      Pagination.css
      Pagination.js
      Search.css
      Search.js
      Table.css
      Table.js
      TableCells.css
      TableCells.js
    vendor/
      ResponsiveCellHeaders.js
    App.css
    App.js
    config.js
    fetchData.js
    index.css
    index.js
    logo.svg
    mag-grey.svg
```

For the project to build, **these files must exist with exact filenames**:

* `public/index.html` is the page template;
* `src/index.js` is the JavaScript entry point.
